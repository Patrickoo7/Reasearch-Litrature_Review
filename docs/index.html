About Research Reproducer

Research Reproducer is a powerful tool designed to help researchers automatically find, setup, and run code from research papers. Simply provide a paper source and the tool handles everything else.

Quick Start (CLI)

# Install
pip install -e .

# Reproduce a paper
research-reproduce reproduce 2301.12345

# Launch web interface
research-reproduce web
